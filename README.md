## Data Sources
- **Raw and clinical metadata** downloaded from GEO: GSE39688
- This cohort consists of serum IgG profiling data generated using high-density protein microarrays.
- Protein array signal intensities were log2-transformed, quantile normalized, and median-centered as described in the original study.
